# Virtualenv

`pip install virtualenv`  

`virtualenv -p {which python} {env_name}`  
* `-p` : choose python interpreter 

## e.g. 

`virtualenv -p /usr/bin/python3 myenv`
`source ./myenv/bin/activate`

`deactivate`
